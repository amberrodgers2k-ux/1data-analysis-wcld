# WCLD Data Analysis Project

## Overview
This project analyzes a large-scale dataset (1.5M+ records) to identify patterns and predictive insights.

## What I Did
- Processed and analyzed over 1.5M records
- Built predictive models (LightGBM, Random Forest - conceptual application)
- Evaluated bias across demographic groups
- Translated complex findings into clear, usable insights

## Focus
- Data-driven decision making
- Ethical AI awareness
- Turning complex data into actionable insights

## Tools & Concepts
- Data analysis
- Machine learning concepts
- Structured data thinking

## Outcome
Improved understanding of large-scale data patterns and highlighted the importance of fairness and bias considerations in predictive modeling.
